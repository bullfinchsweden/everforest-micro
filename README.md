Everforest for the Micro text editor.

https://micro-editor.github.io/

https://github.com/sainnhe/everforest

1. Put everforest.micro in ~/.config/micro/colorschemes (or create a new folder named colorschemes if non-existing)
2. Add export MICRO_TRUECOLOR=1 to the shell rc file (.zshrc)
3. Open the Micro text editor, press control+e and type set colorscheme everforest and press enter
4. Done.
